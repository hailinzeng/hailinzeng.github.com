## Overview 

### What is Jekyll?

To start Zookeeper, you need Configuration file in **conf/zoo/cfg**

    tickTime=2000 
    dataDir=/var/lib/zookeeper
    clientPort=2181


Now, you can start Zookeeper:

    bin/zkServer.sh start
